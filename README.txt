Central Chic Warsaw 5.0

Wdrożenie Netlify:
1. Wejdź na netlify.com i zaloguj się.
2. Add new site -> Deploy manually.
3. Przeciągnij cały folder central_chic_v5 albo plik ZIP po rozpakowaniu.
4. Otwórz link https://... w Safari na iPhonie.
5. Udostępnij -> Do ekranu początkowego.
6. Uruchom ikonę i kliknij „Skonfiguruj Face ID”.

Uwaga techniczna:
- Face ID/passkeys działa tylko na HTTPS.
- Ta wersja jest statyczna i zapisuje passkey na urządzeniu. To dobre do prywatnego panelu, ale nie zastępuje pełnego backendu z weryfikacją serwerową.
